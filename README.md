# MediAgent ─ 病院業務 自律AIエージェント（デモ）

院長秘書・総務業務を兼任する**自律型AIエージェント**の動作を、目で見てわかる形で可視化したWebアプリです。
現場で発生した状況（トリガー）に対し、**ReActフレームワーク**で自律的に思考し、最終決定だけは人間が承認する **Human-in-the-Loop** 設計になっています。

## 思考プロセス（ReAct）

1. **状況分析（Thought）** ── 起きた問題と、予測されるリスク・影響を分析
2. **自律アクション（Action）** ── 指示されなくても「今やるべきこと」を決定
3. **実行内容（Execution）** ── メール文面・発注申請書などの具体的アウトプットを作成
4. **完了報告（Result）** ── 人間への事後報告と、最終確認（承認/修正）を求める

## 収録シナリオ（3つの現場トラブル）

| トリガー | 内容 |
|---|---|
| 📅 院長のスケジュール衝突 | 医師会の緊急役員会と製薬会社A社面談がダブルブッキング → A社へ日程変更を自律交渉 |
| ❄️ ナースステーションのエアコン故障 | 看護師の異音連絡 → 保守業者へ最短修理を要請、看護師へ一次返答 |
| 📦 コピー用紙の在庫低下 | 在庫センサー検知＋健診需要を予測 → 15箱の発注申請を自動作成 |

## 技術スタック

- Next.js 16（App Router）/ React 19 / TypeScript
- Tailwind CSS v4
- ログイン不要・APIキー不要の**フロントエンド完結型デモ**

## ローカル実行

```bash
npm install
npm run dev      # http://localhost:3000
```

## ビルド

```bash
npm run build && npm run start
```
